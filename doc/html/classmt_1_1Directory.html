Detailed Description

A specialized File that acts as a container for other File objects.

  • Inherits from mt::File to provide a composite structure, allowing directories to be treated as files while managing a collection of entries.

◆ operator-=()

Directory & mt::Directory::operator-= ( std::shared_ptr< File >  entry)

Removes an entry from the directory and deletes it from the filesystem.

  • This operator performs two actions:
  1. Physical: Calls unlink() on the file's path to remove it from the disk.
  2. Logical: Erases the shared pointer from the internal m_entries vector.
  • Note
    If the entry is a directory, unlink may fail (POSIX requires rmdir).
    Warning
    This is a destructive operation and cannot be undone.
  • Parameters
    entryThe shared pointer to the File or Directory to be removed.
    Returns
    Directory& Reference to this directory for method chaining.
